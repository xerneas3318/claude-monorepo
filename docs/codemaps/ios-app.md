## Build

```bash
cd apps/ios-app
# requires xcodegen (brew install xcodegen)
./bootstrap.sh
open ClaudePlanner.xcodeproj
```

`bootstrap.sh` runs `xcodegen generate` from `project.yml` вЂ” that file is the source of truth for the Xcode project, not `project.pbxproj`. Edit `project.yml` and re-run `bootstrap.sh` to change sources/dependencies/signing.

`Config.local.xcconfig` is per-machine (bundle id, dev team); the `.example` shows the format.

## Talk в†’ relay handshake

`Talk/TalkSettings.swift` stores the relay base URL (`http://<host>:8787` in dev, https in prod). `ClaudeClient` attaches `Authorization: Bearer <Firebase ID token>` and POSTs JSON `{text, history}`. Reply is `{text}`; passed straight to `TTSService.speak()`.
